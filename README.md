# dark-swal_TH (1.0)
## พัฒนาโดย POPODevTeam
### โปรเจค dark-swal_TH พัฒนาเพื่อปรับปรุง SweetAlert 1.0 ให้เป็นโหมดมืด หรือ Dark Mode ให้ผู้ใช้สบายตา ไม่แสบตา ต่างจาก SweetAlert เดิมที่จะเป็นสีขาว ซึ่งจะทำให้แสบตา ไม่สะดวกสบาย และลำบากในการใช้งาน
### ทั้งนี้จึงอยากให้ผู้พัฒนาซอฟต์แวร์ นิสิต นักศึกษา ได้นำโค้ด(Code) นี้ไปใช้ เพื่อที่จะปรับปรุงหน้าตา SweetAlert ของเราให้เป็นโหมดมืด (Dark Mode) ครับ😊
